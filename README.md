/# VagaManager

This is a job management application that allows companies to post job openings and candidates to apply for them.

## Features

*   **Company:**
    *   Register and authenticate.
    *   Create and manage job openings.
*   **Candidate:**
    *   Register and authenticate.
    *   View and apply for job openings.

## Technologies

*   Java
*   Spring Boot
*   Spring Web
*   Spring Security
*   JPA
*   Maven
*   H2 Database (for testing)

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/vagaManager.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd vagaManager
    ```
3.  Run the application:
    ```bash
    ./mvnw spring-boot:run
    ```
4.  The application will be available at `http://localhost:8080`.
